

# Weather_app
A web application for getting weather information based on location.

# Features:
  ### User can search their city
  ### Weather data based on the user's location
  ### Map based on location picker
  ### 7-days weather data
  ### Preloader

### Tech Stack: 
  HTML | CSS | JavaScript
